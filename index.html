Inventory Management System

Overview

This Python project uses tkinter for GUI and demonstrates file handling, database operations, and interactive design.

Scripts & Their Functions

1. dashboard.py

Main dashboard of the system. Contains buttons, labels, images, and live updates.

Enables navigation to employee, supplier, category, products, and sales modules for CRUD operations.

2. employee.py

Handles employee information management. Features include adding, editing, deleting, and searching employees by email, name, or contact.

3. supplier.py

Manages supplier records. Allows CRUD operations and searching by invoice number.

4. product.py

Tracks product information, availability, and supports search by category, supplier, or product name.

5. category.py

Defines product categories. Allows users to add or delete categories (e.g., IPhone â†’ Phone).

6. sales.py

Displays and stores sales data linked to invoice numbers. Useful for billing history tracking.

7. create_db.py

Initializes the SQLite database. Must be run before launching dashboard.py to avoid errors.

8. billing.py

Handles billing operations, customer details, selected products, pricing, and discounts.

Includes a calculator to compute the total bill. All buttons and interactions are functionalized.

Usage Steps

  1. Run create_db.py to initialize the database.
  2. Run dashboard.py to launch the main interface.
  3. Use the menu to manage employees, suppliers, products, and billing.
  4. Use sales.py to view saved bills.

Pre-Requisites

  • pip install time
  • pip install pil
  • pip install sqlite3
  • pip install os

Configuration

  • Create a folder named images to store images used in the app.
  • Create a folder named bill to store generated bills.
  • Run the Python scripts in the correct sequence for proper functioning.
